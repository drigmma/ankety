# Анкетный Telegram-бот для лагеря

Бот на Python + aiogram 3 для заполнения родительских и детских анкет с отправкой данных в Google Sheets.

## Запуск

1. Скопируйте `.env.example` в `.env` и заполните значения.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите бота:

```bash
python bot.py
```
